Internet Exploder 7

Thank goodness Internet Explorer is gone.

I designed this error simulator in 2015, out of sheer frustration with having to work with Internet Explorer's lack of standards. Testing on Chrome and Firefox, everything looks fine... then coming to Internet Explorer and the navigation elements are inexplicably halfway down the page. Slightly ironically, I had to correct this simulator before uploading - since it was designed using IE8, it now displayed incorrectly in modern browsers.

Originally only displayed in IE 9 and below, using the tag <!- -[IF IE]>. Now virtually no one would see it, since less than 1% of people still use IE 7 where errors like this occur. Thank goodness for that!
